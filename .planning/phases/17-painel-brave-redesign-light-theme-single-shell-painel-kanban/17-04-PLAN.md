<objective>
Build the Kanban board: 5 horizontal-scroll stage columns of draggable record cards. Cards (the `RecordCard` component) reuse the canonical StageBadge score-band taxonomy and show the design's fields (chip, score, name, UF, município, source, duplicate flag, ⚠ falha + ↺ reprocessar). Purpose: A faithful, tested board surface with a clean drag-event contract.
Output: `RecordCard.tsx` + `PainelBoard.tsx` + tests.
</objective>
