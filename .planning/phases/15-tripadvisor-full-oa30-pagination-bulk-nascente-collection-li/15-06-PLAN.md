<objective>
Add the NEW bulk national ingest path that resolves the operator-locked blocker: write all-Brazil
attractions to Nascente WITHOUT a parent destino, deriving uf + município from the national geocode +
IBGE resolution. This is a DISTINCT path (`_ingest_one_bulk` + `produce_paginated`) — the existing
per-UF `_ingest_one` parent-linkage path stays byte-for-byte intact (CONTEXT locked: bulk is a new
path, not a mutation of the old contract).

`produce_paginated` drives `fetch_attractions_paginated` (15-04), ingests each card via `_ingest_one_bulk`,
commits PER PAGE (so a mid-run 403 leaves durable records + an accurate resume point — Pitfall 3), and
records progress + per-card error count per completed page.

Purpose: Make the ~80% reuse actually reach Nascente for a national attraction-only run.
Output: bulk ingest method + paginated producer + offline tests proving Nascente rows > 0 without a parent.
</objective>
