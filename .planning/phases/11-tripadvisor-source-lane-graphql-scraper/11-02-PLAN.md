<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| GraphQL response → TripAdvisorDestinoPayload | Untrusted external data parsed at Pydantic boundary |
| review fields → TripAdvisorReviewSignals | LGPD enforcement point: extra="forbid" blocks author/text from entering Nascente |
| route_by_score → mar_ready flag | Privilege marker; must only be True for TA attractions meeting the bar |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-02-01 | Information Disclosure | TripAdvisorReviewSignals LGPD boundary | mitigate | model_config=ConfigDict(extra="forbid") rejects any field not explicitly declared; test asserts ValidationError on author/text; no review text ever reaches Nascente payload |
| T-11-02-02 | Tampering | mar_ready flag in route_by_score | mitigate | Flag is only True when entity_type=="attraction" AND canonical_key.startswith("tripadvisor:") AND atualidade≥bar AND corroboracao≥bar; explicit False for all other paths; unit test asserts False for mtur/places sources |
| T-11-02-03 | Denial of Service | IBGE CSV load on every request | accept | CSV is 5570 rows loaded once at producer construction; load_ibge_csv cached by caller (producer holds ibge_records list); no per-request file I/O |
| T-11-02-04 | Tampering | parent_rio_id injection | mitigate | parent_rio_id comes from destino_rio_map (dict populated from same sweep's store_raw results, not from user input); quarantine if absent, never infer |
</threat_model>
