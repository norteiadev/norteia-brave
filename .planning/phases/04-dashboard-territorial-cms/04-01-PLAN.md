| browser → BFF → FastAPI | Operator token crosses; the FastAPI Bearer dep is the server-side gate on the read surface |
| dashboard mutation → write-to-production | validate/approve sets validação-humana=100 and pushes to norteia-api — highest-value boundary |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Spoofing | require_bearer | mitigate | hmac.compare_digest constant-time compare; fail-closed on unset token; 401 before any DB work (mirrors require_steward) |
| T-04-02 | Elevation of Privilege | either-or mutation guard | mitigate | either-or still requires ONE valid secret; unset Bearer token must not let Bearer-presented requests pass; steward path unchanged |
| T-04-03 | Information Disclosure | bearer_token / steward secret | mitigate | secret never logged; DashboardConfig uses env_prefix-only (no alias, CR-02 secret-shadowing rule) |
| T-04-04 | Tampering | settings/deps config | mitigate | fail-closed default `Field(default="")` — an unset token rejects all callers |
| T-04-SC | Tampering | python deps (none added) | accept | no new package installs in this plan; existing pinned deps only |
