| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-05-01 | Tampering | MturSeedIngest payload construction | accept | Payload values come from the bundled CSV (public data). No user input in this path. | T-02-05-02 | Denial of Service | produce(uf) with large UF dataset | mitigate | CSV is read once per produce call; no network. If CSV has thousands of rows, processing is linear. Batch size can be limited per UF query. |
