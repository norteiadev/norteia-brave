# Phase 17.1 Plan 05: Varreduras Frontend Summary

Typed runs-history client + its MSW contract mirror + the `PainelVarreduras` table view: runs by Fonte/Profundidade/UF with colored status pills, 7-day stat cards, and a real ↺ Falhas reprocess.

## What Was Built

**Task 1 — runs-api client + MSW handler (commit `830224e`)**
- `dashboard/lib/runs-api.ts`: `RunItem` / `RunsResponse` / `RunReprocessResult` interfaces mirroring `brave/api/routers/runs.py` field-for-field (`id, started_at, ended_at, ufs, source, depth, total, synced, failed, status`); `runsKeys` factory on the `['runs']` prefix; `fetchRuns({uf,source,depth})` and `reprocessRun(id)` through `apiFetch` (relative `api/v1/...`, BFF Bearer); plus 7-day window helpers (`RUNS_WINDOW_HOURS`, `recentRuns`, `totalRuns/totalSynced/totalFailed`, `formatCount`) mirroring the cost-api total/format idiom.
- `dashboard/mocks/handlers/runs.ts`: factories `runsListSuccess()`, `runsListEmpty()`, `runsListError()`, `runsReprocessSuccess()` at the double-prefixed `http://localhost:3000/api/api/v1/runs/...`; payloads typed against the lib interfaces (the A5 contract mirror). Not registered in the empty `mocks/handlers/index.ts` barrel — follows the established per-suite `server.use()` pattern (mirrors cost.ts/dedup.ts).

**Task 2 — PainelVarreduras view + test (commit `6742998`)**
- `dashboard/components/painel/PainelVarreduras.tsx` (`"use client"`): `useQuery(fetchRuns)` driven by Fonte + Profundidade Seg filters (server-side) and a UF Seg group derived from the loaded set (client-side over `ufs`); runs table with columns Início (started_at) / UF / Fonte / Profundidade / Total / Sincr. / Falhas / Status. Status maps to a colored pill (concluido/parcial/falha/running) with the design's oklch literals. Per-row ↺ Falhas wired to `useMutation(reprocessRun)` + sonner toast + `invalidateQueries(['runs'])`, disabled when `failed==0`. 7-day SummaryCards reuse the runs-api window helpers. Empty state. Pure `--painel-*` token styling.
- `__tests__/PainelVarreduras.test.tsx`: 4 offline Vitest (rows + columns + status pills; 7-day stat cards; ↺ Falhas fires the real reprocess PATCH; empty state).
