## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client.py → TripAdvisor HTTPS | Outbound GraphQL POST; response is untrusted — parse with Pydantic in later plan |
| config.proxy_url → residential proxy | Proxy URL contains credentials (if used); must not appear in logs |
| Redis brave:ta:session | Session cookie jar stored in Redis; must not be logged at INFO level |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-01-01 | Information Disclosure | TripAdvisorConfig.proxy_url logging | mitigate | proxy_url must never be emitted in structlog calls; use `proxy_url="[redacted]"` in any log line that references it |
| T-11-01-02 | Information Disclosure | brave:ta:session cookie jar in Redis | mitigate | session key stored encrypted-by-convention (no author PII, but DataDome cookies could be replayed); TTL enforced (config.session_ttl, default 1800s); never logged |
| T-11-01-03 | Elevation of Privilege | Playwright headless browser execution | mitigate | Playwright lazy-imported; never reachable from API path; only invoked from Celery sweep task behind engine auth |
| T-11-01-SC | Tampering | playwright package install | mitigate | scraper dep group is optional + separate; slopcheck gate required before first `pip install playwright`; not in CI |
</threat_model>
