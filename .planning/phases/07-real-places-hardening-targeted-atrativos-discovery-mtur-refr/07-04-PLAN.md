    1. Create scripts/mtur_xlsx_to_csv.py:

    Module docstring: "Convert Portaria MTUR 9/2025 XLSX export to the Norteia Mtur CSV schema.\n\nThe Mapa do Turismo Brasileiro XLSX is downloaded manually from:\n  https://www.mapa.turismo.gov.br/mapa/init.html\nSelect 'Município Categorizado Detalhado Excel' → year 2025, all UFs → Download.\n\nUsage:\n  pip install openpyxl  # if not installed\n  python scripts/mtur_xlsx_to_csv.py municipios_mtur_2025.xlsx\n  # writes: data/mtur/municipios_mtur_2025.csv\n  # dry-run (print first 5 rows only):\n  python scripts/mtur_xlsx_to_csv.py municipios_mtur_2025.xlsx --dry-run\n"

    Imports: from __future__ import annotations; import csv; import sys; import os; from pathlib import Path

    Constants:
    OUTPUT_SCHEMA = ["co_municipio", "no_municipio", "sg_uf", "categoria", "no_regiao_turistica"]

    COLUMN_CANDIDATES — dict mapping each output column to a list of candidate XLSX column name variants (case-insensitive match):
    - "co_municipio": ["co_municipio", "codigo_ibge", "cod_ibge", "codibge", "co_municipio_ibge", "codigo_municipio"]
    - "no_municipio": ["no_municipio", "nome_municipio", "municipio", "nome"]
    - "sg_uf": ["sg_uf", "uf", "estado", "sigla_uf"]
    - "categoria": ["categoria", "categorizacao", "cat"]
    - "no_regiao_turistica": ["no_regiao_turistica", "regiao_turistica", "regiao", "nome_regiao_turistica"]

    Function _detect_columns(headers: list[str]) -> dict[str, str | None]:
    - Returns {output_col: actual_xlsx_col_name} by matching headers (stripped, lower) against COLUMN_CANDIDATES
    - Prints detected mapping to stdout for operator verification
    - Returns None for any column not found

    Function convert(xlsx_path: Path, output_path: Path, dry_run: bool = False) -> None:
    - Try import openpyxl; if ImportError: print error + "pip install openpyxl"; sys.exit(1)
    - wb = openpyxl.load_workbook(xlsx_path, read_only=True, data_only=True)
    - ws = wb.active
    - Read first row as headers (strip whitespace from each cell value, handle None)
    - Call _detect_columns(headers) → mapping
    - Print detected mapping
    - If any required column is None (co_municipio, no_municipio, sg_uf, categoria): print warning + "Check XLSX columns manually"; list detected headers; sys.exit(1)
    - Iterate rows (skip header row); for each row: extract values by column index; write to CSV
    - If dry_run: print first 5 rows, do not write file, print "DRY RUN — no file written"
    - If not dry_run: write to output_path with csv.writer, newline='', encoding='utf-8'; print f"Written {row_count} rows to {output_path}"
    - Handle no_regiao_turistica: if None in mapping, write empty string for that column

    main() function:
    - sys.argv[1] is xlsx path (required); sys.argv[2] is output path (optional, defaults to data/mtur/municipios_mtur_2025.csv relative to repo root)
    - --dry-run flag support (check "--dry-run" in sys.argv)
    - Validate xlsx_path exists; print error if not
    - Call convert(Path(xlsx_path), Path(output_path), dry_run)

    if __name__ == "__main__": main()
