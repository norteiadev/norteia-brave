```python
from brave.core.models import mask_phone
def _safe_normalized(normalized: dict | None) -> dict:
    n = dict(normalized or {})
    contacts = n.get("contacts")
    if isinstance(contacts, dict) and "phone_e164" in contacts:
        contacts = dict(contacts)
        contacts["phone_masked"] = mask_phone(contacts.pop("phone_e164", None))
        n["contacts"] = contacts
    return n
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser/operator → BFF Route Handler | Operator Bearer token validated here; 401 before forwarding to FastAPI |
| BFF → FastAPI /api/v1/destinos + /atrativos | Service Bearer (server-held secret) injected; new CMS endpoints must verify it |
| PATCH body → normalized field update | User-supplied JSON dict merged into rio.normalized; must exclude phone_e164 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01 | Spoofing | GET /api/v1/destinos, GET /api/v1/atrativos (read) | mitigate | `require_bearer` on all read endpoints — constant-time HMAC, fail-closed; 401 if missing or wrong |
| T-08-02 | Tampering | PATCH /api/v1/destinos/{id}/promote, /descarte, /reprocess | mitigate | `require_steward_or_bearer` — either-or guard; delegates to existing service functions, no raw SQL writes |
| T-08-03 | Tampering | PATCH /api/v1/atrativos/{id}/advance expected_state bypass | mitigate | advance_sub_state idempotency guard checks expected_state; 409 on mismatch; Pydantic AdvanceBody constrains body shape |
| T-08-04 | Information Disclosure | phone_e164 in atrativo list/detail responses | mitigate | _safe_normalized applied on every atrativo serialization path; phone_e164 key explicitly excluded from /edit body processing |
| T-08-05 | Tampering | PATCH /edit body injects phone_e164 back into normalized | mitigate | /edit handler filters phone_e164 key from body.fields before merging into normalized; flag_modified enforces SQLAlchemy dirty tracking |
| T-08-SC | Tampering | npm/pip installs | accept | No new packages in this plan |
